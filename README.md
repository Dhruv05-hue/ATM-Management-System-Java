# ATM Management System

## Overview

ATM Management System is a desktop-based banking application developed using Java Swing and MySQL. The system allows users to perform common ATM operations such as deposits, withdrawals, balance enquiry, PIN changes, and transaction history management through a graphical user interface.

## Features

* User Authentication
* Deposit Money
* Withdraw Money
* Fast Cash
* Balance Enquiry
* Mini Statement
* PIN Change
* Transaction History
* MySQL Database Integration

## Technologies Used

### Frontend

* Java Swing
* JFrame

### Backend

* MySQL

### Development Tools

* Eclipse IDE
* MySQL Workbench
